Encabezados (h1 - h6)

Las etiquetas <h1> to <h6> se utilizan para definir encabezados HTML.
<h1> Define el título más importante.
<h6> Define el encabezado menos importante.
Nota: Utilice solo uno <h1> por página; este debe representar el título/asunto principal de toda la página. Además, no se salte los niveles de encabezado: comience con <h1>, luego use <h2>, y así sucesivamente.

Ejemplo de uso

<h1> encabezado </h1>

Imagenes <img>

Esta etiqueta es utilizada para Introducir imagenenes dentro de los sitios web.

Ejemplo de uso

Listas odenadas <ol>

Esta etiqueta es utilizada para introducir listas ordenadas, osea listas con numeracion en cada uno de sus elementos.

Ejemplo de uso

<ol> </ol>

Listas desordenadas <ul>

Esta etiqueta es utilizada para introducir listas desordenadas, osea listas simples sin importar el orden de los elementos.

Ejemplo de uso

<ul> </ul>

List Item <li>

Esta etiqueta es utilizada para Introducir elementos a una lista, sin sin importar si es ordena y/o desordenada

Ejemplo de uso

<ul> <li> soy un elemento </li> </ul>
